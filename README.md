# Txt to JSON parser

## Steps

-   txt file format should be

```
1. Question
    1. Answer one
    2. Answer two
    3. Answer three
    4. Answer four
    5. Answer five
2. Question two
    a. Answer a
    b. Answer b
... etc.
```

-   copy file into project directory under name `master_test.txt`
-   run `python main.py`
-   get file `out.json` from project dir
